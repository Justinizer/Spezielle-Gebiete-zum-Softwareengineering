Save picture to disk.

This function saves the given picture (cv::Mat image) in to the given path as a pgm-image. A picture name is created with the current time + "img" + the number of the pictures in this picture set, because it can be necessary to save pictures in the same time. (For example: If there is more than one person)

Parameters
[in]frameThe picture (Mat object) which should be saved.
[in]pathEnter a path to save picture.
[in]nThe number of picture. Default is 0.
