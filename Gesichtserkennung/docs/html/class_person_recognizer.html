Detailed Description

PersonRecognizer class to recognize persons from a given image.

The PersonRecognizer class recognizes person or persons from given set of images by using LBPH by openCV.

This constructor reads first of all the pictures from files and creates users according to file names. It crreates a face recognizer, in this case with LBPH pattern. Then, it trains the face recognizer with the pictures from the files. Then, the software is ready to recognize.

◆ recognizePersonFromFace()

User * PersonRecognizer::recognizePersonFromFace ( cv::Mat  faceFrame)

recognize the person

This function takes the detected face and updates the confidence number of the user.

Confidence number shows us that how similar the person from the faceFrame to our user (K-nearest neighbor). When confidence <= "RecognizedConfidence", the person is recognized. When confiden > "UnrecognizedConfidence", the person is not one of the users that we know "RecognizedConfidence" and "UnrecognizedConfidence" numbers are defined in jsettings.json
